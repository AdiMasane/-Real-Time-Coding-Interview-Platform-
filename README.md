# -Real-Time-Coding-Interview-Platform-

A full-stack real-time coding interview platform that enables live coding, video calls, and automated evaluation. Built to simulate real technical interviews with secure code execution and real-time collaboration.

🚀 Features

🔐 Secure authentication with Clerk

💻 VSCode-like live code editor

🎥 1-on-1 video interviews (mic/camera, screen share, recording)

💬 Real-time chat messaging

⚙️ Secure code execution in isolated environment

🎯 Auto evaluation with test cases (pass/fail feedback)

🎉 Confetti on success + failure notifications

🧩 Practice problems page (solo mode)

🔒 Room locking (max 2 participants)

🧠 Background jobs using Inngest

⚡ Fast data fetching & caching via TanStack Query

🛠️ Tech Stack

Frontend: React.js, TanStack Query
Backend: Node.js, Express.js
Database: MongoDB
Auth: Clerk
Real-Time: Stream Video & Chat
Async Jobs: Inngest
